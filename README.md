## 环境要求
需要 Node 6.0+ 环境

## 安装  
``` shell
$ git clone https://github.com/MercyLong/Cloud-Music.git
$ npm install
```
## 运行
``` shell
$ npm run dev

```

服务器默认占用8080和3000两个端口,其中3000端口用于启动网易云音乐Node Api服务,8080是为了承载页面（感谢提供API接口的作者[@Binaryify](https://github.com/Binaryify/NeteaseCloudMusicApi)）
